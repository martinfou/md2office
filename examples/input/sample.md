# Sample Document - MD2Office Converter

This is a sample markdown document to demonstrate the **md2office** converter capabilities.

## Introduction

The md2office converter transforms Microsoft Copilot-generated markdown into professional Office documents. It supports conversion to:

- **Word** (.docx) format
- **PowerPoint** (.pptx) format  
- **PDF** format

## Features

### Markdown Support

The converter handles all standard markdown elements:

#### Headings

All six heading levels are supported, from H1 to H6.

#### Lists

**Unordered Lists:**
- First item
- Second item
  - Nested item 2.1
  - Nested item 2.2
- Third item

**Ordered Lists:**
1. First numbered item
2. Second numbered item
3. Third numbered item

### Advanced Features

#### Table of Contents

When using the `--toc` option, a table of contents is automatically generated from document headings.

#### Page Breaks

Use `--page-breaks` to insert page breaks at major section boundaries (H1/H2 headings).

#### Bookmarks

PDF and Word documents include bookmarks for easy navigation.

## Usage Examples

### Basic Conversion

```bash
# Convert to Word
md2office --word document.md

# Convert to PowerPoint
md2office --powerpoint document.md

# Convert to PDF
md2office --pdf document.md
```

### Advanced Options

```bash
# Convert to all formats with custom output directory
md2office --all --output ./output document.md

# Use professional style preset
md2office --word --style professional document.md

# Generate table of contents
md2office --word --toc document.md
```

## Configuration

You can create a `.md2office.json` configuration file:

```json
{
  "defaultFormats": ["word", "powerpoint"],
  "outputDirectory": "./output",
  "style": "professional",
  "pageBreaks": true,
  "tableOfContents": true
}
```
